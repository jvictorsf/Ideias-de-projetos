### Sistema de Gestão Escolar
Neste sistema, o seu foco é o gerenciamento de uma escola, ou seja, cadastrar alunos, professores e outros funcionários. Neste sistema você poderá utilizar banco de dados e criar novas implementações.

Alguns exemplos de implementação de um Sistema de Gestão Escolar:
 - CRUD (Create, Read, Update e Delete) de alunos, professores e funcionários;
 - Gestão de notas (Calcular notas dos alunos);
 - Sistema de ponto de funcionário;
 - Impressão de boletim escolar; Dentre outras coisas.

### Sistema de Gestão Bancária
Crie um programa que simulará um sistema bancário simples com crédito e débito em contas bancárias.
O programa deverá:
 - Criar conta poupança e corrente
 - Depositar e retirar dinheiro
 - Ver saldo
 - Encerrar conta

**Avançando com o projeto:** Desenvolva e implemente um banco de dados para armazenar as informações das contas.

**Opcional:** Desenvolva um sistema de impressão de comprovante de saque, e então o comprovante, ou deve ser salvo em PDF ou imprimido.
<hr>

### Sistema de Gestão de Agenda
Neste projeto, o sistema deverá operar como uma agenda, o sistema deve ter funções como:
 - Escrever uma página;
 - Ler uma página;
 - Pesquisar por frases;
 - Pesquisar página por data;

**Implementações opcionais:**
 - Implementar a um banco de dados;
 - Implementar recurso de senha (senha, chave-frase);
 - Marcação de texto;
 - Inserção de imagem.

### Sistema de Gestão de Cobrança de Cafetaria
Crie um sistema típico de cobraça de clientes para uma cafeteria. O programa deverá possuir:
 - Uma lista de itens no menu de acordo com seu tipo (lanches, refeição, bebida etc);
 - e seu respectivo preço;
 - qualquer desconto oferecido;
 - modo de pagamento.

Você poderá utilizar banco de dados para uma melhor implementação.

Você também poderá criar outras funções, como:
 - Cadastrar o cliente;
 - Dar desconto para o cliente aniversáriante etc.

### Sistema de Gestão de Alunos
Crie um sistema de gerenciamento de alunos. O sistema deverá cadastrar os alunos, fazer as médias, aprovações de alunos, impressão de boletins etc.

Utilize banco de dados para ter um melhor desempenho do sistema.

### Sistema de Gestão de Reserva de Hotel
É um projeto de reserva de hotel, deve efetuar a reserva de um quarto e o pagamento, mas o projeto deverá ter a função de ver o quarto, ter desconto, ver se o quarto está livre etc.

### Sistema de Gestão de Aluguel de Veículos
O programa deverá possuir menus separados para administradores e outros usuários. Deverá haver também métodos para calcular a tarifa com base no tempo e na distância, incluindo a exibição de detalhes do carro, disponibilidade etc.

### Sistema de Gestão de Folha de Pagamento
Crie um programa que faça o calculo da folha de pagamento.

## Jogos
### Joguinho da cobra
Ah, o famoso joguinho da cobrinha, quem nunca jogou? Este é um ótimo método de aplicar seus conhecimentos.

O jogo da cobrinha é fazer com que a cobra coma as "maçãs" e a cada maçã comida ela cresce e o jogador ganha ponto, caso ela bata nela mesmo ou nos limites, o jogador perde.

O jogo base é apenas isso, mas vocÊ poderá implementar outras coisas, como:
 - Sistema de Ranking;
 - Sistema para mudar a "skin" da cobra; dentre outras coisas que você imaginar.

### Pedra, Papel, Tesoura
Acho que todos conhecemos o famoso jogo "pedra, papel e tesoura". Bom o jogo consistem em vencer, papel vence pedra, pedra vence tesoura e tesoura vence papel.

Além do jogo base, você pode adicionar a função de classificação de ranking.

### Adivinhe o número
Neste jogo, você deve tentar advinha o número que a máquina escolheu e conforme vocÊ acerta, você ganha um ponto e no final da rodada, veja se você ou a máquina ganhou.

### Jogo de Perguntas
Este é um projeto simples baseado em perguntas e respsotas, no qual você criará um banco de perguntas para vários tópicos, como política, história, geografia, esportes, filmes etc. Conforme você acerta as perguntas, seu score (pontuação) aumento, também deverá ter um ranking de quais jogadores pontuarão mais.

**Avançando:** Adicione também algumas outras funções, como:
 - Tempo fixo para responder a cada pergunta;
 - Níveis de dificuldade;

## Outros
### Calculadora comum
Crie uma calculadora comum, no ínicio pode ser uma que faz simples contas básicas. Conforme vai avançando seus estudos você pode ir melhorando-a.

Por exemplo:
 - você pode integrar uma mémoria nela, para que guarde resultados anteriores;
 - usar mais de um operador;
 - implementar uma interface gráfica; dentre outras coisas que você pode imaginar. 

### Calculadora de IMC
Este projeto tratasse de uma calculadora de IMC (Índice de Massa Corporal).

Você pode ver a interpretação (o "esquema")do IMC [clicando aqui](https://www.programasaudefacil.com.br/calculadora-de-imc)


### Conversores de unidades
Neste projeto você criará conversores, você pode criar diversos tipos de conversores, de temperatura, de metragem, de pesagem etc.

Caso tenha mais de um conversor, você pode criar um menu para que o usuário decida o que quer converter.

### Sistema de Reserva de Ônibus
Crie um sistema que simule um sistema de reserva de ônibus. O sistema terá funções como:
 - Cadastro de ônibus:
 	- Tipo do ônibus;
 	- Placa;
 	- Quantidade de assentos;
 	- Carga total de bagagem;
 	- Preço da bagagem por Kilo;
 - Cadastro de viagem:
 	- Cidade;
 - Reservar assento:
 	- Origem
 	- Destino
 	- Distância entre origem e destino:
 	- Preço da viagem
 	- Imprimir todos os dados acima + um valor de ID
 - Cancelar assento reservado
 	- ID
 	- Confirmar ou não

### Sistema de Login e Registro
O projeto envolve um processo de registro do usuário solicitando nome de usuário e senha. após o registro bem-sucedido, um arquivo de usuário é criado com as credenciais. Caso o usuário não exista, ao efetuar o login, será apresentado um erro.
